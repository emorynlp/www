The ELIT Project

What does ELIT stand for?

It stands for "Emory Langauge and Information Toolkit".
It also stands for "Evolution of Langauge and Information Technology".
The ELIT project brings the latest natural language processing (NLP) technology to researchers and developers. The primary focus of this project is to build an end-to-end NLP framework that is robust across various domains and scalable for big data analysis. ELIT is under the Apache 2 license and developed by the Emory NLP with an active collaboration with the GluonNLP.
